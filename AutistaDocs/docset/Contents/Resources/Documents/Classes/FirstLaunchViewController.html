scrollViewDidScroll:

Handling scrollViewDidScroll event. We don’t want a “feedback loop” between the UIPageControl and the scroll delegate in which a scroll event generated from the user hitting the page control triggers updates from the delegate method. We use a boolean to disable the delegate logic when the page control is used.
